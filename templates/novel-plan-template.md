## 第0阶段: 研究与准备

### 目标
收集创作所需的背景知识和参考资料。

### 研究任务
- [ ] 研究小说类型的经典作品和流行趋势
- [ ] 收集时代背景相关的历史/文化资料
- [ ] 研究目标读者群体的阅读偏好
- [ ] 分析同类型成功作品的特点
- [ ] 收集专业术语和行业知识(如果需要)

### 输出
- `research.md` - 研究发现和参考资料汇总

**关卡**: 研究完成,可以开始详细设计

---

## 第1阶段: 详细设计

### 目标
创建完整的角色、世界观和情节大纲。

### 设计任务

#### 1.1 角色设定 (`characters.md`)
- [ ] 主要角色(每个角色包含):
  - 基本信息(姓名、年龄、外貌)
  - 性格特点和心理特征
  - 背景故事和成长经历
  - 核心动机和目标
  - 角色弧光(成长轨迹)
  - 与其他角色的关系
- [ ] 次要角色(简化版设定)
- [ ] 角色关系图

#### 1.2 世界观设定 (`worldview.md`)
- [ ] 时空背景(时代、地点、社会环境)
- [ ] 世界规则(如果是幻想类型)
- [ ] 社会结构和权力体系
- [ ] 文化习俗和价值观
- [ ] 重要地点描述
- [ ] 关键物品/系统设定(如果需要)

#### 1.3 情节大纲 (`outline.md`)
- [ ] 整体故事结构(起承转合)
- [ ] 按情节优先级(P1, P2, P3)组织
- [ ] 每个情节的详细展开:
  - 情节目标和冲突
  - 关键场景列表
  - 情节转折点
  - 情感曲线
- [ ] 章节划分和节奏安排
- [ ] 伏笔和呼应设计

#### 1.4 写作指南 (`writing-guide.md`)
- [ ] 文风示例和要求
- [ ] 叙事节奏控制
- [ ] 对话风格指南
- [ ] 场景描写技巧
- [ ] 需要避免的写作陷阱
- [ ] 质量检查清单

### 输出
- `characters.md` - 完整的角色设定
- `worldview.md` - 世界观和背景设定
- `outline.md` - 详细的情节大纲
- `writing-guide.md` - 写作风格指南

**关卡**: 设计文档完成并通过审查,可以开始任务分解

---

## 第2阶段: 任务分解

### 目标
将大纲分解为可执行的写作任务。

### 任务组织原则
- 按情节优先级(P1, P2, P3)组织任务
- 每个情节独立可完成
- 每章作为一个独立任务
- 标记可以并行创作的章节

### 输出
- `tasks.md` - 详细的写作任务列表(由 `/novelkit.tasks` 命令生成)

**关卡**: 任务列表完成,可以开始实际创作

---

## 第3阶段: 创作执行

### 目标
按照任务列表完成实际写作。

### 执行原则
- 按优先级顺序创作(P1 → P2 → P3)
- 每完成一章进行自我审查
- 保持与角色设定和世界观的一致性
- 遵循写作指南中的风格要求
- 定期检查情节进展和节奏

### 质量控制
- [ ] 每章完成后进行文字校对
- [ ] 检查与前文的连贯性
- [ ] 验证角色行为符合设定
- [ ] 确认情节推进合理
- [ ] 检查伏笔和呼应

**关卡**: 初稿完成,进入修改阶段

---

## 第4阶段: 修改与润色

### 目标
提升作品质量,准备发布。

### 修改任务
- [ ] 整体结构调整
- [ ] 情节逻辑优化
- [ ] 角色塑造深化
- [ ] 文字润色和优化
- [ ] 删除冗余内容
- [ ] 强化高潮部分
- [ ] 完善开头和结尾

### 最终检查
- [ ] 运行 `writing-guide.md` 中的质量检查清单
- [ ] 确认所有 [需要澄清] 标记已解决
- [ ] 验证满足所有成功标准
- [ ] 检查篇幅是否符合预期
- [ ] 确认适合目标读者群体

**关卡**: 作品完成,准备发布

## 风险与缓解

### 创作风险
- **风险**: 情节过于复杂,难以控制
  - **缓解**: 严格遵守简洁性关卡,限制角色和情节线数量
  
- **风险**: 角色行为不一致
  - **缓解**: 创建详细的角色设定文档,写作时经常参考
  
- **风险**: 节奏拖沓或过快
  - **缓解**: 在大纲阶段规划好情感曲线,定期检查

- **风险**: 偏离目标读者喜好
  - **缓解**: 在研究阶段充分了解读者,定期征求反馈

### 进度风险
- **风险**: 创作进度延迟
  - **缓解**: 设定合理的每日/每周字数目标,保持稳定节奏
  
- **风险**: 创作灵感枯竭
  - **缓解**: 提前准备充足的素材和参考,保持灵感库
